# contacts_list

# 📇 Contact Manager — Python Tkinter + SQLite

Une application graphique simple et intuitive permettant de gérer une liste de contacts.  
Développée avec **Python 3**, **Tkinter** pour l’interface graphique et **SQLite** comme base de données locale.

---

## 🚀 Fonctionnalités

- ✔️ Interface graphique moderne basée sur Tkinter  
- ✔️ Ajout d’un nouveau contact (Nom, Prénom, Email, Contact)  
- ✔️ Affichage des contacts dans un `Treeview`  
- ✔️ Suppression d’un contact sélectionné  
- ✔️ Fonction de modification prête à être intégrée  
- ✔️ Base de données SQLite intégrée automatiquement  
- ✔️ Design simple, propre et lisible
